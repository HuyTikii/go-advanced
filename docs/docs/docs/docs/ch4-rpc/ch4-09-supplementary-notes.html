4.9 Ghi chú bổ sung

4.9. Ghi chú thêm

Chúng ta nên đọc thêm các tài liệu và ví dụ được trang chủ Protobuf và gRPC cung cấp. Trong chương này chúng ta tập trung vào RPC của thư viện chuẩn Go và framework gRPC dựa trên Protobuf. Song song đó là tìm hiểu cách tự mình tùy chỉnh framework RPC. Hiện tại, cộng đồng Open source có rất nhiều framework RPC riêng biệt, cũng như các hệ thống RPC để tùy biến trên các hệ thống phân tán. Người dùng nên chọn đúng công cụ theo nhu cầu thực tế của mình.
